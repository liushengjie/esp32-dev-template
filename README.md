# esp32-dev-template
基于ESP32-S3整理各类开发模块示例模版,方便后续组合使用
1.  TFT液晶显示开发模版 tft_display--TFT屏幕128*160,ST7735S驱动
    -   TFT_eSPI移植开发模版 
    -   LVGL开发模版
    -   Gui-Guider集成开发模版
2.  WIFI模块开发模版 wifi
    -   esp32_wifi直接链接ssid
    -   通过WifiMuti存储ssid并进行连接
    -   使用WifiManager通过手机配置进行wifi连接
    -   通过HTTPClient进行HTTP访问
3.  LED模块开发模版
    -   使用fastLED控制WS2812B
